Provides classes to demonstrate opb-plsql use in a JSE environment.

Database set-up

To run all of the demo's you need to create a database account.
  1. To create the DB account (opb_jse_demo). e.g. Run the following commands via SQL*Plus;
    • CREATE USER opb_jse_demo DEFAULT TABLESPACE users IDENTIFIED BY weak_pw;
    • GRANT DBA TO opb_jse_demo;
  2. Build the demo SQL;
    • Run src/main/sql/build.sql from this project as opb_jse_demo.
The default DB connection details are (see applicationContext.xml);
  • url=jdbc:oracle:thin:@//localhost:1521/xe
  • user=opb_jse_demo
  • password=weak_pw
This URL will work if you have a local copy of Oracle XE running.
If you would like to use a different URL or user, please edit the configuration of the oracleDataSource bean in src/main/resources/applicationContext.xml.
