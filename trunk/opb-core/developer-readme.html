Database set-up

To run the unit tests, you need to set-up a database account; The opb_test account is set-up with all Opb features enabled (such as session tracking and the ability to create user messages).
  1. Create the DB account (opb_test). e.g. Run the following commands via SQL*Plus;
    • CREATE USER opb_test DEFAULT TABLESPACE users IDENTIFIED BY weak_pw;
    • GRANT DBA TO opb_test;
  3. Build the objects needed to test opb-core;
    • Run src/main/resources/sql/build.sql from the opb-plsql project as opb_test.
  4. Build the objects needed to test opb-plsql;
    • Run src/test/resources/plsql/build.sql from the opb-plsql project as opb_test.
The default DB connection details are (see helpers.TestHelper);
  • url=jdbc:oracle:thin:@//localhost:1521/xe
  • user=opb_test
  • password=weak_pw
This URL will work if you have a local copy of Oracle XE running.
If you would like to use a different URL or user, please edit helpers.TestHelper.
